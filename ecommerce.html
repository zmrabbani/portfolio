Ecommerce

Simple Ecommerce website using Laravel 9 as the framework.

Demo

This e-commerce project is the final project for e-learning website development with Coding Studio. The project is a simple e-commerce platform using Laravel 9 as its framework. The website functions like a typical e-commerce platform, where users can view items for sale, complete purchases, and more. Users are required to log in or register to make transactions. Upon selecting a product, its details are displayed, and users can choose the quantity before adding it to their cart. After proceeding to checkout and submitting the receipt of payment, the items in the cart are removed. Users can also view a list of their orders.

The website interface differs for admin users, who can delete and edit products, confirm payments, and add new products. Admins have access to the complete list of orders from all users.

All user data, product catalogs, cart contents, transaction records, and more are stored in the phpMyAdmin database. You can see the results of this project in the video. If you have any further questions, feel free to contact me using the provided contact information.
